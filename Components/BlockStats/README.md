# BlockStats

Stats block with a large number + optional icon on the left (pinned mid-screen on scroll) and corresponding text blocks on the right. As you scroll, the left stat updates to match whichever text block is at the center of the viewport.

## Usage

Add via Page Components. Each repeater row has: **Number/Stat**, **Icon** (optional), **Title**, **Description**. The left column sticks at 50vh and shows the stat for the block currently at the viewport center.
